Questo repository contiene il sito di H4SH BOMB. L'immagine in alto è la versione aggiornata del logo; dopo aver pushato queste modifiche su GitHub, vai su "Settings → Social preview" del repository e, se necessario, ricarica o carica manualmente l'immagine preview per forzare l'aggiornamento.

Comandi suggeriti per pubblicare le modifiche:

```bash
git add README.md
git commit -m "Add README with updated logo for GitHub preview"
git push origin main
```

Se la preview rimane vecchia: rimuovi e riaggiungi l'immagine nella sezione "Social preview" delle impostazioni del repository o attendi qualche minuto perché la cache si propaghi.
